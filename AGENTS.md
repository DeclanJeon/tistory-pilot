# Tistory Agbrowse Project

이 프로젝트는 Tistory 블로그 자동 발행 및 관리를 위한 도구 모음이다.

## 스킬

### tistory-blog
한국어 기술 블로그 포스트 작성 및 발행 스킬.

**트리거 키워드:**
- `tistory`, `블로그`, `포스트`, `글쓰기`, `발행`
- `blog`, `post`, `tistory publish`
- `$tistory-blog` 명시적 호출

**사용 시점:**
- Tistory 블로그 글을 작성할 때
- 블로그 포스트를 발행할 때
- 글의 레이아웃/스타일을 개선할 때
- 이미지가 포함된 기술 글을 쓸 때

**규칙:**
1. `~/.codex/skills/tistory-blog/SKILL.md`를 먼저 읽는다
2. HTML 템플릿과 스타일 규칙을 따른다
3. AI 패턴 금지 (`한 줄 요약`, `먼저 핵심만 보자` 등)
4. 마크다운 문법 금지 (Tistory는 HTML 렌더링)
5. 이모지 섹션 헤더 사용
6. 노란 인사이트 박스 + 파란 정보 박스 활용
7. 블록인용으로 핵심 인용 표시
8. 다크 코드 참조 박스로 읽은 코드 표시
